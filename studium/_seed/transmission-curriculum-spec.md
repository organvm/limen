## Larger Reading System

What you are describing should not be treated as "reading books." It is a **transmission curriculum**:
epic, scripture, language, handwriting, translation, commentary, and music.

The right structure is:

> **Read the text → copy the original script → translate a small unit → compare translations → write one interpretive note → listen to one composition → log the resonance.**

## Core Western / Abrahamic / Medieval Sequence

| Phase | Text               | Language / Script Layer          | Main Question                           |
| ----- | ------------------ | -------------------------------- | --------------------------------------- |
| 1     | *Iliad*            | Homeric Greek / Greek alphabet   | What is wrath?                          |
| 2     | *Odyssey*          | Homeric Greek                    | What is return?                         |
| 3     | *Aeneid*           | Latin                            | What is empire built from?              |
| 4     | *Metamorphoses*    | Latin                            | What is form/change?                    |
| 5     | *Divine Comedy*    | Italian                          | What is ordered ascent?                 |
| 6     | Bible / Torah      | Hebrew + Greek + Latin reception | What is covenant, law, creation, exile? |
| 7     | Qur'an             | Arabic                           | What is revelation as recitation?       |
| 8     | *Beowulf*          | Old English                      | What is heroic life under doom?         |
| 9     | *Canterbury Tales* | Middle English                   | What is social speech?                  |

For Greek and Latin source work, **Perseus** (texts like Vergil's *Aeneid*, Homer's *Odyssey*, other Greek/Latin), also pointing toward **Scaife** and **Open Greek and Latin**.
For the Torah/Tanakh layer, **Sefaria** (Tanakh as Torah, Nevi'im, Ketuvim — texts, translations, commentaries, interconnections).
For the Qur'an layer, **Quran.com** (Arabic text + translations, tafsir, recitations, word-by-word tools, search).

## Eastern / Asian Additions

Do not add these as "extra world literature." Add them as **counter-systems** to the Greek-Roman-Abrahamic-medieval line.

### Tier 1: Immediate Additions

| Text                     | Tradition        | Script / Language                                  | Why It Belongs                                                              |
| ------------------------ | ---------------- | -------------------------------------------------- | --------------------------------------------------------------------------- |
| *Bhagavad Gita*          | Sanskrit / Hindu | Devanagari + Sanskrit                              | War, duty, action, metaphysics; direct counterpoint to Achilles and Arjuna  |
| *Ramayana*               | Sanskrit epic    | Devanagari + Sanskrit                              | Exile, kingship, fidelity, divine-human order                               |
| *Mahabharata* selections | Sanskrit epic    | Devanagari + Sanskrit                              | A civilization-scale epic, more encyclopedic than the *Iliad*               |
| *Dhammapada*             | Buddhist         | Pali, optionally Devanagari or Sinhala/Thai script | Compact ethical-poetic counterweight to heroic violence                     |
| *Dao De Jing*            | Chinese Daoist   | Classical Chinese / Hanzi                          | Anti-heroic, anti-force, anti-overassertion                                 |
| *Zhuangzi*               | Chinese Daoist   | Classical Chinese / Hanzi                          | Transformation, uselessness, dream, paradox                                 |
| *Analects*               | Confucian        | Classical Chinese / Hanzi                          | Ritual, conduct, social ethics, governance                                  |
| *Tale of Genji*          | Japanese         | Kana + Kanji                                       | Interior court epic; psychology, aesthetics, impermanence                   |
| *Tale of the Heike*      | Japanese         | Japanese                                           | War, fall, impermanence; closer to *Iliad* than *Genji* in martial register |
| *Shahnameh*              | Persian          | Persian script                                     | Heroic epic; kingship, fate, lineage, empire                                |

**GRETIL** for Sanskrit/Pali source work (machine-readable Indian-language texts incl. *Mahabharata*, *Bhagavadgita*, *Ramayana*, Pali).
**Chinese Text Project (ctext.org)** for Classical Chinese (*Analects*, *Mengzi*, *Zhuangzi*, *Dao De Jing*, *Shiji*, *Journey to the West*, *Hong Lou Meng*) — 30k+ titles, 5B+ characters.

## Daily Study Protocol

Keep it finite. Do not let this become an infinite research swamp.

### Daily Minimum

| Step            |      Time | Output                                               |
| --------------- | --------: | ---------------------------------------------------- |
| Primary reading | 20–40 min | Read one scene, not necessarily one chapter          |
| Script copying  |    10 min | Handwrite 3–5 lines / verses / sentences             |
| Translation     |    15 min | Produce literal translation + freer literary version |
| Commentary      |    10 min | One paragraph: what changed in your understanding?   |
| Music           |  5–15 min | One composition matched to the day's dominant force  |
| Ledger          |     3 min | Log text, passage, script, composition, motif        |

```text
LITERARY TRANSLATION:

meter / syntax / repetition / epithet / image / law / genealogy / invocation

ONE INTERTEXT:
How does this speak to another text in the sequence?

Why this composition fits:

KEEP / REPLACE:
```

## Script-Learning Order

Do not try to learn every language equally. Use **script first, grammar second, translation third**.

| Order | Script / Language         | Use                                          |
| ----: | ------------------------- | -------------------------------------------- |
|     1 | Greek alphabet            | Homer: names, epithets, repeated phrases     |
|     2 | Latin                     | Vergil, Ovid, medieval reception             |
|     3 | Italian                   | Dante in original terza rima                 |
|     4 | Hebrew                    | Torah, Psalms, prophetic diction             |
|     5 | Arabic                    | Qur'an as recitation, root-pattern awareness |
|     6 | Old English               | *Beowulf*, alliteration, compound diction    |
|     7 | Devanagari / Sanskrit     | *Gita*, *Ramayana*, *Mahabharata*            |
|     8 | Classical Chinese / Hanzi | *Dao De Jing*, *Analects*, *Zhuangzi*        |
|     9 | Kana + Kanji              | Japanese court and war literature            |
|    10 | Persian script            | *Shahnameh*, Attar, Hafez, Rumi              |

The key is not fluency. The key is **physical contact with the text**. Handwriting slows you down enough to notice structure.

## Daily Classical Composition System

For each day, assign the passage one dominant category:

| Textual Force       | Musical Requirement                        |
| ------------------- | ------------------------------------------ |
| Wrath               | rhythmic pressure, unresolved harmony      |
| Plague              | descending motion, dark orchestration      |
| Supplication        | suspended cadence, exposed melodic line    |
| Divine intervention | sudden color shift, brass/strings contrast |
| Exile               | wandering tonality, open intervals         |
| Return              | circular form, recognition motif           |
| Revelation          | stillness, chant, verticality              |
| Law                 | formal clarity, repetition, architecture   |
| Comedy              | dance rhythm, courtly exaggeration         |
| Grief               | slow tempo, inward line                    |
| Metamorphosis       | changing orchestral color, fluid harmony   |
| Ascent              | rising sequence, luminous orchestration    |

For *Iliad* Book I, the dominant arc is:
```text
wrath → plague → quarrel → restraint → withdrawal → supplication → divine comedy
```

The daily composition is not background music. It is a **second commentary system**. The handwriting is not decorative. It is a **third commentary system**. Translation is the place where the book stops being an object and becomes an encounter.

## Working Thesis for the Whole Project

You are not just reading "great books." You are studying how civilizations encode:
```text
violence · law · memory · fate · sacrifice · speech · revelation · empire · desire · death · return · transformation · salvation · comedy
```
