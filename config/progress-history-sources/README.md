# Progress history source adapters

Historical selection discovers these adapters at runtime. Each JSON file maps
one registered source owner document into the generic history leaf contract.
Source IDs, document paths, identity/state fields, and terminal semantics are
data; adding or renaming a source does not require a Python catalog edit.

Runtime/private facts are listed before bounded tracked fallbacks. A truncated,
missing, stale, or non-exhaustive document remains history coverage debt.
