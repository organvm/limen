# Podcast OS GitHub Recomposition Package

This package translates the connected GitHub audit into a canonical product architecture.

## Files

- `PODCAST_OS_GITHUB_RECOMPOSITION_BLUEPRINT.md` — narrative architecture and build order.
- `repo_capability_matrix.csv` — source-by-source retain/extract/adapt/archive decisions.
- `domain_kernel.yaml` — canonical entities, relationship classes, state machine, events, and hard rules.
- `integration_manifest.yaml` — source repositories mapped to target services/packages.
- `target_architecture.mmd` — Mermaid flow diagram.

## Important boundary

No GitHub repository was edited, merged, archived, deleted, or otherwise mutated during this audit.
