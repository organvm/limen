# OpenCode Capacity Blocked

Agent is unreachable via `limen.capacity.agent_status`.

Command:
```
PYTHONPATH=cli/src python3 -c 'from limen.capacity import agent_status; print(agent_status("opencode"))'
```

Output:
```
{'agent': 'opencode', 'kind': 'local-cli', 'reachable': False, 'detail': 'opencode not found', 'command': ['opencode']}
```

Fix: The `opencode` CLI must be installed globally (e.g., via `npm install -g opencode-ai`) or the path specified via `LIMEN_OPENCODE_BIN`.
