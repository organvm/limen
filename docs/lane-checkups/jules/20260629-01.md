# Lane Checkup: jules

Date: 2026-06-29
Task: CAPFILL-jules-20260629-01

## Blocker

The `jules` binary is missing from the environment PATH. This prevents the `jules` lane from becoming productive, as the capacity probe (`agent_status("jules")`) reports the agent as unreachable and blocks dispatch.

## Evidence

Running the capacity probe programmatically:

```bash
PYTHONPATH=cli/src python3 -c 'from limen.capacity import agent_status; print(agent_status("jules"))'
```

Output:

```python
{'agent': 'jules', 'kind': 'cloud-cli', 'reachable': False, 'detail': 'jules not found', 'command': ['jules']}
```

Confirming the binary is not found:

```bash
which jules
# Output is empty
```

## Resolution

A human must install the `jules` CLI tool into the system PATH, or configure `LIMEN_JULES_BIN` to point to the installed executable.
