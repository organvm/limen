# Insight

1. **Throughput is real; convergence is weak.** An 88.08% terminal PR yield coexists with a 90.6%
   reopen-to-dispatch event ratio. The fleet can land code, but it revisits too many obligations.
2. **The denominator is winning.** Terminal stock gained 179 tasks while nonterminal stock gained
   106; closure percentage still fell because 285 tasks entered the board.
3. **Evidence is not completion.** Prompt evidence coverage reached 94.5%, while strict terminal
   prompt authority remains unsealed. Any combined progress number would be materially misleading.
4. **Provider capacity is not the main bottleneck.** The latest receipt showed substantial Claude,
   Jules, Gemini, and Agy headroom; ownership, predicates, conflicts, and host/resource gates were the
   limiting factors. OpenCode alone was near throttle.
5. **A mega-session becomes coordination debt.** Once handoff, board, main, and active worktrees move
   underneath the session, continuing the same context increases rework. A continuation capsule is a
   control-plane primitive, not clerical ceremony.
