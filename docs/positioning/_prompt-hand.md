# The Prompt Hand

**The most valuable corpus you own is the one you throw away every day — every prompt you type.**

Every prompt is a shot at an idea: a compressed statement of intent, aimed at a system, in the moment you understood the problem best. Then the tab closes and it's gone. The single richest record of how you think — what you asked, in what order, and how often you had to ask twice — evaporates by default. The Prompt Hand is the discipline that refuses to let it.

## The insight almost no one acts on

When you paste the same request a second time, that is not you being repetitive. **A repeated prompt is the system confessing it failed you the first time.** Repetition is not noise to be tolerated — it is a measurable defect signal pointing at the exact place the system dropped your intent. Capture every prompt, and the repetitions become a map of everything still unresolved.

## What it is

A frozen, additive, byte-mirrored capture of every prompt one operator ever typed — never a lost keystroke — decomposed into verifiable atomic units and run through one loop:

**CAPTURE → ATOMIZE → TRIAGE → PRIORITIZE → ROUTE → VERIFY → CLOSE.**

- **Capture** — additive only; every keystroke preserved, nothing overwritten.
- **Atomize** — each prompt split into the smallest units that can be independently verified.
- **Triage** — repetition read as failure-confession; the many restatements of one ask collapse to the canonical ask.
- **Prioritize** — money and architecture float to the top; six dimensions weigh recency, urgency, criticality, specificity, cross-reference density, and trajectory.
- **Route** — each atom goes to its cheapest capable executor; irreversible actions never auto-dispatch.
- **Verify** — a task is never marked done without a cited primary-source anchor.
- **Close** — no unit is allowed to rest unreviewed.

## Four questions held for every atom of intent

For each atomic unit the system keeps four things side by side — which is why nothing gets lost between "you said it" and "it shipped":

- **Prompt** — what was asked
- **Pragma** — what actually exists now
- **Telos** — what it should become
- **Praxis** — what else it could become

## Built to production weight — measured, not asserted

- **5,867** prompts across **523** sessions, captured additively since day one
- decomposed into **24,596** independently verifiable units
- repetition *measured*, not tolerated: two-thirds of the open backlog were restatements of roughly **ten** canonical asks — collapsed to the ten, each resolved once
- verification with teeth: nearly **half** of previously "done" units were reopened when a cited anchor was demanded — because unverified "done" is the most expensive lie a system can tell you

## The mandate

Proceed with all suggestions; logic dictates order; add, never overwrite; verify, never claim; resolve the canonical so the person never has to ask twice.
