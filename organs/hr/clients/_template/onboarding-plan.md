## Pre-Arrival (Day -14 to Day 0)

- [ ] Offer letter signed, background check initiated
- [ ] I-9 / right-to-work documents collected
- [ ] Tax withholding forms (W-4 / state equivalent) collected
- [ ] Benefits enrollment window communicated
- [ ] IT account provisioning requested
- [ ] Workspace / equipment prepared
- [ ] Onboarding buddy assigned

## Day 1 — Orientation

- [ ] Company overview & culture introduction
- [ ] Policy handbook acknowledgment (signed receipt)
- [ ] Emergency contact form completed
- [ ] Payroll enrollment confirmed (direct deposit)
- [ ] Benefits enrollment (if window open)
- [ ] IT equipment issued, accounts activated
- [ ] Office / facility tour (if in-person)

## Week 1 — Integration

- [ ] Role-specific training plan initiated
- [ ] Team introductions completed
- [ ] Tools & systems access verified
- [ ] First 1:1 with manager held
- [ ] Initial goals/deliverables discussed

## 30 Days — Checkpoint

- [ ] Manager check-in: performance, questions, concerns
- [ ] Training progress assessed
- [ ] Access/permissions audit
- [ ] First 30-day goals reviewed

## 60 Days — Deepening

- [ ] Role competencies assessment
- [ ] Styx opt-in discussion (if applicable — consent-first)
- [ ] Ongoing development plan drafted

## 90 Days — Transition to Steady State

- [ ] Performance expectations confirmed
- [ ] Regular review cadence established
- [ ] Onboarding plan closed; transition to performance framework
