# Education Organ (edu-organism)

This is the Education Organ. It is not merely a collection of classes or a set of grading scripts. It is a unified mechanism designed to build, scale, and deploy an alternative educational infrastructure. 

The organ operates simultaneously at two levels:

1. **[The Macro Face (MACRO-FACE.md)](./MACRO-FACE.md)**: The theoretical and strategic spine. The Alternative-Education Thesis. This is the demand-side engine that acknowledges the collapse of trust in legacy institutions and builds the rhetorical machine and alternative credentialing network required to capture the exodus. It is a blueprint for a billion-dollar spine.

2. **[The Micro Face (MICRO-FACE.md)](./MICRO-FACE.md)**: The praxis. The grounded, operational instances proving the model in the real world today. We run three primary instances on our universal curriculum Kernel:
   - **ENC1101 (The Institutional Bridge):** Hyper-automated administration of a legacy master-course, freeing all instructor bandwidth for high-leverage feedback.
   - **Derek's Narrative Program (The Private Mentor Model):** A bespoke, rigorous, one-on-one apprenticeship culminating in the creation of an original graphic novel.
   - **The Homeschool Engine (Adaptive Personal Syllabus):** Our flagship AI-personalized curriculum engine that rewrites content for the individual and generates professional artifacts (the "Wings" framework) instead of just grades.

The code and deep assets for this organ reside in the `organvm/edu-organism` monolith. This directory serves as the public-facing platform and strategic map for the organ within the Limen registry.
